## Working Summary

AI Company OS is a general-purpose AI company operating system. Its first version must prioritize:

- Human Root authority
- Agent registry
- Skill registry
- Workflow engine
- Task state machine
- Permission engine
- Approval center
- Risk engine
- Append-only audit log
- Memory
- Knowledge Base
- Basic dashboard
- Tests for safety and workflow invariants

Do not implement platform abuse, credential theft, phishing, captcha bypass, attack tooling, malicious automation, automatic money movement, audit deletion, risk shutdown, or Root permission modification.
